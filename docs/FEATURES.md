# Daily-cam 아이 곁에 - 기능 명세서

## 🎯 핵심 가치 제안

기존 홈캠을 AI 안전 모니터링 시스템으로 업그레이드하여, 부모님들이 영유아의 안전을 실시간으로 확인하고 데이터 기반 안전 조치를 취할 수 있도록 지원합니다.

## 📊 비즈니스 모델

### 구독 플랜
- **무료 플랜**: 1개 카메라, 기본 알림
- **프리미엄 플랜** (월 29,900원): 무제한 카메라, AI 분석, 일일 리포트
- **기업 플랜**: 어린이집/유치원용 대량 구독

## 🔥 주요 기능

### 1. 홈캠 연동
**목표**: 기존 홈캠을 쉽게 연결하고 안전 구역을 설정

#### 기능 상세
- **카메라 연결 방법**:
  - RTSP URL 직접 입력
  - Wi-Fi QR 코드 스캔
  - 제조사별 자동 인식
  - 지원 브랜드: 대부분의 IP 카메라

- **세이프존 설정**:
  - 드래그로 안전 구역 그리기
  - 다중 세이프존 지원
  - 구역별 이름 지정
  - 시간대별 구역 변경

- **데드존 설정**:
  - 위험 구역 지정 (주방, 계단 등)
  - 접근 시 즉시 알림
  - 위험도 레벨 설정
  - 부모님 부재 시 자동 활성화

### 2. 실시간 모니터링
**목표**: AI가 실시간으로 아이의 행동을 분석하고 위험을 감지

#### 기능 상세
- **AI 실시간 분석**:
  - 객체 감지 (아이, 위험 물체)
  - 행동 패턴 분석
  - 낙상 감지
  - 울음 소리 인식

- **즉시 알림**:
  - 데드존 접근 (높음)
  - 위험 행동 감지 (중간)
  - 활동 변화 (낮음)
  - 알림 방식: 푸시, 이메일, SMS

- **멀티 카메라 뷰**:
  - 최대 4개 동시 시청
  - 자동 카메라 전환
  - 화면 분할 커스터마이징

### 3. 일일 리포트
**목표**: 하루의 안전 상황을 종합적으로 분석하여 제공

#### 기능 상세
- **AI 한줄평**:
  - GPT-4 기반 자연어 요약
  - 주요 이벤트 강조
  - 긍정적 피드백 포함
  - 부모님 맞춤 톤앤매너

- **위험도 분석**:
  - 시간대별 위험 발생
  - 위험 유형별 분류
  - 재발 패턴 감지
  - 우선순위 정렬

- **즉시 실행 리스트**:
  - AI 추천 안전 조치
  - 예상 비용 및 난이도
  - 구매 링크 제공
  - 설치 가이드 영상

- **리포트 공유**:
  - PDF 다운로드
  - 이메일 전송
  - 가족 구성원 공유
  - 의사/전문가 공유

### 4. 데이터 분석 & 히트맵
**목표**: 장기 데이터를 시각화하여 패턴 발견

#### 기능 상세
- **공간 히트맵**:
  - 위험 발생 빈도 지도
  - 활동 집중 구역
  - 시간대별 변화
  - 3D 시각화 옵션

- **시간 히트맵**:
  - 시간대별 위험도
  - 요일별 패턴
  - 계절별 변화
  - 예측 분석

- **트렌드 분석**:
  - 안전도 개선 추이
  - 위험 감소율
  - 행동 패턴 변화
  - 성장 단계별 비교

- **데이터 내보내기**:
  - CSV 형식
  - PDF 리포트
  - 의료 기관용 양식
  - API 연동

### 5. 추가 기능

#### 5.1 음성 알림
- 위험 감지 시 음성 경고
- 양방향 통신 (부모 ↔ 아이)

- 다국어 지원

- 느린 동작 재생

#### 5.3 성장 기록
- 발달 단계 추적
- 첫 걸음, 첫 말 등 기록
- 월별 성장 리포트
- 사진/영상 앨범

#### 5.4 커뮤니티
- 부모님 커뮤니티
- 안전 팁 공유
- 전문가 Q&A
- 제품 추천

## 🎨 UI/UX 특징

### 디자인 원칙
1. **직관성**: 기술에 익숙하지 않은 부모님도 쉽게 사용
2. **명확성**: 위험 상황을 명확하게 전달
3. **안정감**: 따뜻하고 신뢰감 있는 디자인
4. **효율성**: 빠른 정보 접근

### 색상 시스템
- 🟢 **안전**: 초록색 (세이프존, 정상 상태)
- 🔴 **위험**: 빨간색 (데드존, 긴급 상황)
- 🟡 **주의**: 노란색 (경고, 주의 필요)
- 🔵 **정보**: 파란색 (일반 정보, 액션)

### 반응형 디자인
- 데스크톱: 풀 기능
- 태블릿: 최적화된 레이아웃
- 모바일: 핵심 기능 중심
