- **니즈**: 회사에서도 아이 안전 확인, 조부모님께 안전 가이드 제공
- **페인포인트**: 기존 홈캠은 녹화만 되고, 위험 상황 실시간 알림 없음

### 페르소나 2: 박민수 (45세, 자영업)
- **상황**: 3세 딸, 집에서 일하며 아이 돌봄 병행
- **니즈**: 일하면서도 아이 안전 모니터링, 위험 구역 자동 감지
- **페인포인트**: 계속 아이를 확인하느라 일에 집중 어려움

## 📱 주요 사용 시나리오

### 시나리오 1: 첫 설정 (온보딩)

#### Step 1: 회원가입 및 프로필 설정
```
1. 이메일/소셜 로그인
2. 아이 정보 입력 (이름, 나이)
3. 구독 플랜 선택 (무료 체험 7일)
```

#### Step 2: 카메라 연동
```
1. "홈캠 연동" 메뉴 선택
2. 카메라 추가 버튼 클릭
3. RTSP URL 입력 또는 Wi-Fi 연결
4. 카메라 위치 지정 (거실, 아이방 등)
5. 연결 테스트
```

#### Step 3: 안전 구역 설정
```
1. 실시간 카메라 화면 확인
2. "세이프존 그리기" 선택
3. 드래그하여 안전한 놀이 공간 지정
4. "데드존 그리기" 선택
5. 주방, 계단 등 위험 구역 지정
6. 저장 완료
```

### 시나리오 2: 일상적 사용

#### 아침 (출근 전)
```
08:00 - 앱 실행, 카메라 상태 확인
08:30 - 출근하며 실시간 모니터링 ON
09:00 - 조부모님 도착 알림 확인
```

#### 낮 (회사에서)
```
11:30 - [알림] "주방 데드존 접근 감지"
       → 실시간 영상 확인
       → 조부모님께 전화로 안내
       
14:00 - 점심시간에 대시보드 확인
       → AI: "오전에는 안전하게 활동 중입니다"
       
15:30 - [알림] "낮잠 시작 감지"
       → 평화로운 시간 확인
```

#### 저녁 (퇴근 후)
```
18:00 - 일일 리포트 확인
       → AI 한줄평 읽기
       → 위험 상황 3건 검토
       → 즉시 실행 리스트 확인
       
19:00 - "주방 안전 게이트 필요" 추천 확인
       → 추천 제품 링크 클릭
       → 온라인 구매
       
21:00 - 다음날을 위한 세이프존 조정
```

### 시나리오 3: 위험 상황 대응

#### 상황: 주방 접근 (높은 위험)
```
[14:25] AI 감지: 아이가 주방 데드존 진입

[14:25:05] 앱 푸시 알림 (긴급)
          "⚠️ 주방 데드존 접근 감지!"
          
[14:25:10] 부모님 앱 확인
          → 실시간 영상 보기
          → 조부모님께 즉시 전화
          
[14:25:30] 조부모님이 아이를 세이프존으로 이동
          
[14:26:00] AI 확인: "아이가 안전 구역으로 돌아왔습니다"
          
[18:00] 일일 리포트에 자동 기록
        → "오후 2시 25분, 주방 접근 1회"
        → "안전 게이트 설치 권장"
```

#### 상황: 낙상 위험 (중간 위험)
```
[10:15] AI 감지: 소파에서 뛰는 행동

[10:15:10] 앱 알림 (주의)
          "⚠️ 낙상 위험 행동 감지"
          
[10:15:20] 부모님 확인
          → 위험도 평가: 중간
          → 조부모님께 문자 전송
          
[10:16:00] 상황 종료
          
[18:00] 일일 리포트 반영
        → "낙상 위험 행동 1회"
        → "소파 주변 매트 설치 권장"
```

### 시나리오 4: 주간 분석 활용

#### 일요일 저녁 (주간 리뷰)
```
20:00 - "데이터 분석" 메뉴 접속
       
20:05 - 주간 트렌드 확인
       → 안전도: 월 85% → 일 93% (개선!)
       → 주방 접근: 주 15회 → 3회 (대폭 감소)
       → 원인: 안전 게이트 설치 효과
       
20:10 - 공간 히트맵 확인
       → 거실 왼쪽 모서리에 위험 집중
       → 해당 위치에 보호대 추가 결정
       
20:15 - 시간 히트맵 확인
       → 오후 2-4시 활동량 최고
       → 해당 시간 더 주의 필요
       
20:20 - PDF 리포트 다운로드
       → 소아과 검진 시 의사에게 공유 예정
```

### 시나리오 5: 설정 및 최적화

#### 월간 점검 (매월 첫째 주)
```
1. 카메라 상태 점검
   → 펌웨어 업데이트
   → 렌즈 청소 알림
   
2. 세이프존/데드존 재설정
   → 아이 성장에 따른 활동 범위 변화
   → 새로운 가구 배치 반영
   
3. 알림 설정 조정
   → 거짓 알림 감소
   → 민감도 조정
   
4. 구독 관리
   → 사용 통계 확인
   → 플랜 유지/변경 결정
```

## 🚀 성공 사례

### Case 1: 안전사고 예방
```
"주방 화상 사고 막았어요"

- 대응: 즉시 알림 → 조부모님 대응

```
"소아과에서 데이터 보고 도움 받았어요"
- 상황: 아이 행동 패턴 상담
